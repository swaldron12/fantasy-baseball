# Legacy Files

This folder keeps older assignment/demo work out of the active fantasy baseball
ML pipeline without deleting it.

## rag_embedding_assignment

Older RAG, chunking, embedding, and local multi-agent demo scripts plus their
generated outputs. These are preserved for reference, but the active project now
focuses on:

```text
pybaseball/Savant data
   ↓
weekly rolling features and splits
   ↓
category-value targets
   ↓
ML projections
```

## static_link_ingestion

The old static CSV URL manifest lives here. Static links are no longer the main
data plan because the project is moving toward pybaseball/Savant pulls.

`config_empty/` is the old root config folder after its manifest was archived.

## notebooks

Empty notebook placeholder folder from the earlier project layout. It is kept
here so the active root stays focused on runnable pipeline code.
